Como subir el rutero

Guia paso a paso para registrar las facturas del dia antes de que el conductor salga a ruta.

⚠ Regla de oro: sin rutero no hay entrega
Cada vez que una ruta salga, el rutero debe estar cargado en el sistema antes de que el conductor inicie.
🚨
Sube ANTES de que salga el conductor
El conductor no puede registrar entregas de facturas que no esten en el sistema. Si ya salio y no esta cargado, debe esperar.
📌
Puedes subir el rutero parcial
No necesitas esperar a que esten todas las facturas facturadas. Sube las que ya tienes y cuando lleguen mas, subes otro archivo con las nuevas.
✓
Si ya subiste una factura, no pasa nada
El sistema detecta automaticamente las facturas que ya existen y solo sube las nuevas. Puedes volver a subir el mismo archivo sin miedo.

Pasos

  1. 1

    Exporta el rutero desde tu sistema

    Descarga el archivo Excel con las facturas del dia. No necesitas esperar a tener todas las facturas listas — puedes subir lo que ya esta facturado y cuando lleguen mas facturas, subes otro archivo con las nuevas.

    📌 Ejemplo de rutero parcial: Si la ruta del Galan sale a las 7 a.m. con 10 facturas, subelas ahora. Si a las 10 a.m. salen 5 mas de BLE, subes esas 5 en un archivo aparte. El sistema las une sin problema.
    📍 Agrega la columna transportador en el archivo y escribe el nombre del repartidor asignado a cada factura (ej: LUIS, FELIPE, CARLOS). Esta columna es obligatoria para que el sistema asigne correctamente cada despacho al conductor.

    Asi debe verse el archivo Excel antes de subirlo

  2. ↓
  3. 2

    Ingresa con tu usuario

    Abre la URL del sistema en tu navegador e ingresa con tu usuario LOM correspondiente.

    Pantalla de ingreso

  4. ↓
  5. 3

    Selecciona el archivo Excel

    Haz clic en el area de carga o arrastra el archivo Excel directamente sobre ella.

    Arrastra el archivo o haz clic

    📌 El archivo debe contener las columnas: # de Factura, Tercero, transportador y Cantidad. Las columnas Sucursal y Fec.Entr. son opcionales.
  6. ↓
  7. 4

    Revisa la vista previa

    Antes de confirmar, el sistema muestra una tabla con todas las filas leidas. Verifica que los datos se vean correctos.

    Sin errores — listo para importar

    Con errores — corrige en Excel y vuelve a subir

    ⚠ Un solo error bloquea toda la subida. Si ves filas marcadas en rojo, corrige el Excel original y vuelve a cargarlo.
  8. ↓
  9. 5

    Confirma la carga

    Si la vista previa no muestra errores, haz clic en el boton Confirmar para importar las facturas al sistema.

    Resultado exitoso

    ✓ Si una factura ya existia, se omite automaticamente sin error. Puedes volver a subir el mismo archivo sin problema.
  10. ↓
  11. 6

    Avisa al conductor y deja que salga

    Confirma que el rutero quedo cargado y notifica al conductor. A partir de este momento puede ingresar a la app y registrar cada entrega.

    🚨 Este paso se hace ANTES de que el conductor salga. Si el conductor ya salio sin que el rutero este cargado, no podra registrar las entregas hasta que alguien lo cargue.
  12. ↻
    Vuelve al Paso 1
    en la proxima ruta

Formato del archivo Excel

Ejemplo de archivo Excel con el formato correcto

Columna Obligatoria Formato Ejemplo
# de Factura SI XXXX-XXXXXXX FVE1-1234567
Tercero SI Texto libre Supermercado Central
transportador SI Texto libre LOM1
Cantidad SI Numero entero 12
Sucursal No Texto libre Norte
Fec.Entr. No Fecha 25/06/2026

Errores comunes

✖ Sin guion en la factura
FVE11234567
Debe incluir el guion: FVE1-1234567
✖ Prefijo con longitud incorrecta
FV-1234567 o FVENTA-1234567
El prefijo debe tener exactamente 4 caracteres.
✖ Numero con longitud incorrecta
FVE1-123456 o FVE1-12345678
El numero debe tener exactamente 7 digitos.
✖ Columna obligatoria faltante
El archivo no tiene la columna # de Factura, Tercero, transportador o Cantidad.
Revisa los encabezados del Excel.
✓ El sistema filtra duplicados automaticamente — si subes facturas ya cargadas antes, las omite sin error. Puedes volver a subir el mismo archivo con total seguridad.
